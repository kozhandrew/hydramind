# 💧 HydraMind – Ваш личный помощник для контроля водного баланса

**HydraMind** — это минималистичное, элегантное и полностью автономное веб-приложение (PWA), созданное для того, чтобы мягко напоминать вам о необходимости пить воду в течение дня.


## ✨ Особенности

*   **Минималистичный дизайн (Glassmorphism)**: Приятный темный интерфейс, который не отвлекает, но радует глаз.
*   **Автономная работа**: Приложение использует **Web Workers**, поэтому таймер не останавливается и не отстает, даже если вкладка браузера неактивна или свернута.
*   **Локально синтезированные звуки**: Все звуки (капля, колокольчик) генерируются через **Web Audio API** в реальном времени. Это означает нулевое потребление трафика на загрузку аудиофайлов.
*   **Progressive Web App (PWA)**: Вы можете установить HydraMind прямо на рабочий стол вашего телефона (Android/iOS) или компьютера в качестве нативного приложения!
*   **Работает без интернета**: После первой загрузки приложение может работать полностью офлайн.

## 🚀 Как использовать?

1. Откройте приложение в браузере.
2. Выберите комфортный для вас **интервал** (частоту напоминаний), например, 10 минут.
3. Выберите **звук оповещения** (его можно предварительно послушать кнопкой "Проверить звук").
4. Нажмите **Старт**.
5. Занимайтесь своими делами! Когда вода в круге "дотечет" до верха, прозвучит сигнал, и таймер автоматически начнется заново.

### Установка на телефон (Android)
1. Откройте сайт приложения в Яндекс Браузере или Chrome.
2. Нажмите на меню браузера (три точки в углу).
3. Выберите **«Установить приложение»** / **«Добавить на главный экран»**.
4. Готово! Запускайте HydraMind прямо с экрана телефона.

## 💻 Установка для разработчиков (Локальный запуск)

Если вы хотите запустить проект у себя на компьютере или модифицировать его:

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/ВашПользователь/hydramind.git
   cd hydramind
   ```

2. **Установите зависимости (Vite):**
   ```bash
   npm install
   ```

3. **Запустите локальный сервер:**
   ```bash
   npm run dev
   ```

Приложение будет доступно по адресу `http://localhost:5173`.

## 🛠 Технологии

*   **HTML5 / Vanilla CSS**
*   **Vanilla JavaScript (ES6)**
*   **Web Audio API** для звуковых эффектов
*   **Web Workers API** для стабильной фоновой работы таймера
*   **Vite** для быстрой сборки и разработки
